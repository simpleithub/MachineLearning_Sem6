# UCS2612-ML Lab (Semester 6)

This repository contains all Machine Learning lab exercises for Semester 6, including Jupyter notebooks and corresponding experiment reports (PDF). Each folder represents one experiment and includes both implementation and documentation.


## Folder Description

- **Ex1** – Supervised learning experiments on:
  - Iris classification  
  - Loan amount prediction  
  - Diabetes prediction  
  - Email spam classification  

- **Ex2** – Contains experiment 2 implementation and report.

- **Ex3** – Contains experiment 3 implementation and report.

- **Ex4** – Contains experiment 4 implementation and report.

## Contents of Each Folder

Each experiment folder includes:
- `.ipynb` – Jupyter Notebook with full code, outputs, and visualizations.
- `.pdf` – Formal lab report for submission.

## Tools & Libraries Used

- Python  
- NumPy, Pandas  
- Matplotlib, Seaborn  
- Scikit-learn  
- Jupyter Notebook / Google Colab

## How to Run

1. Open any `.ipynb` file using Jupyter Notebook or Google Colab.
2. Run cells sequentially to reproduce results.
3. Refer to the corresponding `.pdf` for theory and explanations.
